Building pipeline:
1-> Github repo
2-> add src
3-> add data,model,report directory to .gitignore
4->  Now git add,commit,push

Setup DVC pipeline(withoput params)
5-> create dvc.yaml and add stages to it.
6-> dvc init then do " dvc repro" to test the pipeline automation.
7-> Now git add,commit,push

Setup DVC pipeline (with params)
8-> add params.yaml 
9-> Add the params setup 
10-> "dvc repro" again to test the pipelinne alonng with the params
11-> Now git add,commit,push

Experiment Tracking with DVC :
 1> pip innstall dvcline
 2> add dvclive code block 
 3> "dvc exp run " 
 4> "dvc exp show "
 5> "dvc exp remove {exp-name}"[Remove] | " dvc exp apply {exp-name}"[Reproduce]
6> change params, re-run code(produce new experiments)
7> Now git add, commit,push
